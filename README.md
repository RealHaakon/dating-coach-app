# dating-coach-app

![Dating Coach](Image/Dating%20coach.jpg)

> Agentisk AI-slop. Generert i sin helhet av en AI-agent. Ingen menneskelig tanke, redigering eller kvalitetssikring er lagt ned i dette prosjektet.

## Hva er dette?

En "interaktiv dating coach"-nettside som en AI-agent spytta ut basert på en prompt. Alt innholdet — leksjoner, kode, design, "råd" — er hallucinert maskinell tekst uten rot i virkeligheten. Ikke bruk dette som faktisk dating-råd.

## Leksjoner (AI-generert innhold)

- **Grunnlaget** (begynner) — Selvbevissthet, verdier og hva du ser etter i en partner
- **Første kontakt** (begynner) — Isbrytere, åpningslinjer, kroppsspråk og tilstedeværelse
- **Samtalekunst** (middels) — Aktiv lytting, spørsmålsteknikk og naturlig samtaleflyt
- **Dating-etikette** (middels) — Planlegging av date, betaling, oppfølging og texting
- **Selvtillit** (avansert) — Mindset, håndtere avvisning og vekstmentalitet

## Installasjon

```bash
npm install
npm start
```

Serveren kjører på `http://localhost:3000`.

## Teknologi

- **Backend**: Node.js, Express
- **Templating**: EJS
- **Innhold**: Markdown med YAML-frontmatter
- **Styling**: Custom CSS med mørk/lys-modus
